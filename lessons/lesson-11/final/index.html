In this final you will be building a micro site. This site will be two pages but note that the navigation will have more links. You will need to build the HTML and CSS from scratch. You will have two classes to complete the work. At the end of the last class you will need to send me the html, css and all images needed to have the site work local.

In orders to make the development faster the site should be linked with relative paths so there is no need to use MAMP.

getting started

  1. create a folder (yourname_final)
  2. create 2 html files (history.html and season.html)
  3. create 1 css file (global.css)
  4. create an images folder

Building HTML

THe first thing you want to do to is to build out one of the html files. Once that file is 100% complete you can use the contents of it for the other file.

Its best to get all the HTML in place before you start working on the CSS but know that you might need to make changes to meet the need of the design.

Once you have the HTML built note that any change that you make to one file will need to be made to every other page. You goal is the template done before you worry about adding your content.

Building the CSS

Once the HTMl is done you should start to build the CSS.

Because we do not build the CSS like a printer (start at the top ened at the bottom) we need to work on getting the structure in place (header, footer, nav etc).

Next work on one section of the page like the header or nav and get that in good shape dose not need to be 100% done but the correct colors, fonts and spacing. Add any backgorund image at the end.

Move on to the next section and work on that.

Ones all the sections are done add the finishing touches like background image, hovers and on states.

Keep it simple

There is nothing to crazy going on here. Nothing that we have not done in class.

Keep your code clean, well thought out and organized. Remember to format your code correct from the beginning.

All spacing will be 5px, 10p, 15px etc.
